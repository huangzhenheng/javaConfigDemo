关于我们 > 公司荣誉
公司相继荣获 "中国驰名商标""广东省名牌产品"、"广东省著名商标"、"东莞市质量奖"（东莞市首届政府质量奖，由市政府组织评比颁奖）荣誉称号，成为广东省石材行业唯一同时拥有此四项荣誉的企业。2009年和2012年先后两次获得国家"高新技术企业"称号。

受国家建材局的委托，主笔撰写了《异型装饰石材》系列标准，填补了国内异型加工标准的空白，该标准于1999年3月获准通过并颁布实施。某某建材致力于中 国石材行业的健康发展，多次承接国家的专业研究课题，先后参与起草和制定了《天然花岗石建筑板材》、《天然大理石建筑板材》、《天然饰面石材试验方法》等 国家、行业标准32项。

公司多次荣获中国建筑行业最高奖项-"鲁班奖"，同时是行业内华表杯优质石材装饰工程奖获奖次数最多的企业，并荣获中国房地产开发企业500强首选石材供应商称号。
